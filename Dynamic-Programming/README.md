## Common Topics
- Memoization
- Tabulation
- Optimal substructure
- Overlapping subproblems
- Knapsack problems
- Longest common subsequence

## Example Problems
- Fibonacci Number
- Climbing Stairs
- Coin Change
- Longest Increasing Subsequence
- House Robber
