# Dynamic Programming

Dynamic programming problems and techniques (memoization, tabulation, etc.).

## Problems

| File | Description |
|------|-------------|
| `equal_subset_partition_bitmask.cpp` | Determines whether an array can be split into two subsets with equal sums using bitmask enumeration. |
| `longest_common_subsequence.cpp` | Finds the longest common subsequence of two strings using a rolling DP table. |
| `subsequence_query_checker.cpp` | Given a main sequence and multiple query subsequences, checks whether each query is a subsequence of the main sequence. |
| `vacation_planner.cpp` | Maximises the total score over n days by picking one of three activities per day (no two consecutive days can share the same activity). Classic DP on a 3-state grid. |
